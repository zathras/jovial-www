Bill Foote's List of Web Resources

Here's a fairly random collection of sites I've run into. This list is not actively maintained; most of the links herein are over two years old (as of 10/98).

Non-ancient links

  • This list of RealAudio radio stations is a good way to burn net bandwidth.

OO Resources

  • Object Currents is an on-line journal produced by SIGs (and without the subscription hassles that come with JOOP!)
  • The Comp.object FAQ is an excellent overview of OO.
  • In addition to the Comp.object newsgroup, there's an archive here. See also the Dejanews search engine.
  • A search engine for Object-Oriented Information Sources is maintained by the University of Berne.
  • The On-Line Computing Dictionary is a good terminology reference.
  • The Patterns Home Page contains a number of useful links.
  • Just to be annoying, here's an OO Newsletter in French.
  • Gimme some of that old-time religion, at the Programming Language Critiques page :-)

Eiffel Resources

  • The Eiffel Page.
  • The (other) Eiffel Page :-)
  • ISE's homepage.

Java Resources

  • The Java FAQ is a good place to start.
  • Sun's Official site for Java is another one of the key sites.
  • Yahoo's Java Links can be a good place to go trolling for information.
  • comp.lang.java is pretty useless, due to a hopeless noise/signal ratio. You might want to look at the summaries in Digital Espresso.
  • Gamelan has a good Java directory, if you can stand forms mode :-)
  • JavaWorld is an on-line magazine for the Java community
  • Pause Java is an on-line Java publication in French.
  • Here's a French Java Page.

C++ Resources

  • The C++ FAQ is pretty complete.
  • Masochist? Then you'll love the ISO Committee Draft for C++ and the ANSI C++ Draft.
  • comp.lang.c++ is useless. Try comp.lang.c++.moderated instead.

General Resources

  • ftp://rtfm.mit.edu is where text versions of the FAQ for just about every newsgroup can be found.
  • Sunsite is where most Linux software can be found. This can be a good way to find the source for other platforms (like Solaris), too.
  • Yahoo is a good general index to the Web.
  • Dilbert needs no introduction.
  • Can't afford a dictionary? Try the Hypertext Webster Interface.

Music

  • I'm a big fan of They Might Be Giants.
  • The Cole Wide Web is a great source of information about Cole Porter.
  • Burn that Internet bandwidth! The Cuban Music Page has recordings of Cuban AM radio.
